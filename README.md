# 🧠 플래시 카드 - 지능형 단어 학습 앱

어빙하우스 기억곡선을 적용한 지능형 플래시 카드 앱입니다. 단어를 효과적으로 학습하고 장기 기억으로 전환할 수 있도록 도와줍니다.

## 🚀 배포 방식

이 앱은 **로컬 스토리지**를 사용하여 데이터를 저장합니다. 별도의 데이터베이스 설정 없이 바로 사용 가능합니다.

### Vercel 배포 방법
1. GitHub 저장소에 코드 푸시
2. Vercel에서 GitHub 연결
3. 자동 배포 시작

**데이터베이스 설정 불필요** - 브라우저의 로컬 스토리지를 사용합니다.

## 🎯 주요 기능

### 📚 학습 모드
- **5지 선다형 퀴즈**: 단어를 보고 의미를 선택하는 형태
- **즉시 표시**: 단어와 보기가 함께 나타남 (클릭 불필요)
- **진행률 추적**: 10개 단어 단위로 학습 진도 표시
- **즉시 피드백**: 정답/오답에 대한 시각적 피드백 제공

### 🧠 지능형 복습 시스템
- **어빙하우스 기억곡선**: 최적의 복습 간격 자동 계산
- **우선순위 알고리즘**: 별표 단어, 자주 틀리는 단어 우선 노출
- **연속 정답 처리**: 5번 이상 맞춘 단어는 노출 빈도 감소
- **적응형 학습**: 개인의 학습 패턴에 맞춰 조정

### 🎉 칭찬 시스템
- **정답 시 칭찬**: 랜덤 칭찬 메시지 표시
- **학습 시작 격려**: 학습 시작 시 응원 메시지
- **중간 칭찬**: 50% 진행 시 격려 메시지
- **완료 칭찬**: 성취도에 따른 칭찬 메시지

### ⭐ 단어 관리
- **단어 추가/편집/삭제**: 직관적인 인터페이스
- **별표 표시**: 어려운 단어를 별표로 표시하여 집중 학습
- **파일 가져오기/내보내기**: CSV 형식으로 단어 목록 관리
- **난이도 설정**: 각 단어별 난이도 지정 가능

### 📊 통계 및 분석
- **학습 통계**: 총 단어 수, 별표 단어 수, 평균 점수
- **학습 일수**: 학습한 총 일수 추적
- **최근 학습 기록**: 최근 5개 세션의 점수 및 성과
- **세부 분석**: 맞춘 개수, 소요 시간, 정답률

## 🚀 사용 방법

### 1. 학습 시작하기
1. 메인 화면에서 "학습 시작하기" 버튼 클릭
2. 자동으로 10개의 단어가 선정됨 (어빙하우스 알고리즘 적용)
3. 카드를 클릭하여 단어 확인
4. 5지 선다형 퀴즈 풀기
5. 다음 단어로 진행

### 2. 단어 관리하기
1. 상단 메뉴에서 "단어관리" 클릭
2. "단어 추가" 섹션에서 새 단어 입력
3. 별표 버튼으로 중요 단어 표시
4. 파일 가져오기로 대량의 단어 추가 가능

### 3. 학습 기록 확인
1. 상단 메뉴에서 "통계" 클릭
2. 전체 학습 현황 확인
3. 최근 학습 세션의 점수 확인

## 📱 반응형 디자인
- **PC**: 전체 화면 최적화
- **태블릿**: 적응형 레이아웃
- **모바일**: 터치 친화적 인터페이스

## 🧪 기술적 특징

### 데이터 구조
- **단어 테이블**: 단어, 의미, 난이도, 학습 상태 정보
- **학습 기록 테이블**: 각 단어별 학습 결과 저장
- **세션 점수 테이블**: 학습 세션별 점수 및 통계

### 알고리즘
- **우선순위 계산**: 별표 > 자주 틀리는 단어 > 복습 시간 초과 > 일반 단어
- **복습 간격**: 정답 시 2배 증가 (최대 30일), 오답 시 절반 감소 (최소 1일)
- **연속 정답 처리**: 5번 이상 정답 시 노출 빈도 대폭 감소

## 🎯 학습 효과

### 장기 기억 향상
어빙하우스 기억곡선에 따라 최적의 시점에 복습하여 장기 기억으로 전환

### 효율적 학습
중요한 단어와 어려운 단어에 집중하여 학습 시간 최적화

### 동기부여
실시간 점수와 통계로 학습 동기 유지

## 🔧 개발자용 정보

### 테이블 스키마
- words: 단어 정보 저장
- learning_records: 학습 기록 저장  
- session_scores: 세션별 점수 저장

### 주요 함수
- `getPriorityWords()`: 우선순위별 단어 정렬
- `showQuiz()`: 5지 선다형 퀴즈 생성
- `saveLearningRecord()`: 학습 기록 저장
- `updateWord()`: 단어 정보 업데이트

## 🌐 웹사이트 접속

웹사이트에 접속하여 바로 사용 가능합니다. 별도의 설치나 설정 없이 즉시 학습 시작!

**주요 URL:**
- `/` - 메인 페이지 (학습 시작)
- 내부 라우팅으로 학습 모드, 단어 관리, 통계 페이지 이동
